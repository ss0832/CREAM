# Potential files

Potential files are not included in this example bundle. Place them next to the scripts before running the examples.

Expected files:

```text
examples/eam_alloy_verification/
  Cu01.eam.alloy
  Mishin-Ni-Al-2009.eam.alloy
```

Suggested sources:

- `Cu01.eam.alloy`: NIST Interatomic Potentials Repository entry for the Mishin Cu EAM potential.
- `Mishin-Ni-Al-2009.eam.alloy`: NIST Interatomic Potentials Repository entry for the Purja Pun--Mishin Ni-Al EAM potential.

Do not commit potential files unless their redistribution terms permit it. A safer repository policy is to commit this README plus file hashes for the exact files used in verification.
